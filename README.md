# LAN Clipboard (prototype)

这是一个桌面 Electron + 内置网页服务的剪贴板共享原型，目标是在局域网内共享文件和文本片段，并通过浏览器提供轻量的上传/下载/下载后删除接口。

主要功能（原型）:
- Electron 桌面窗口显示按时间排序的项目列表（文本/文件）。
- 内置 Express 网页服务（默认端口 3000），提供 /api/items, /api/upload, /api/download/:id, /api/delete/:id。
- 网页前端提供上传、下载、下载后删除的简洁界面。

快速开始:

1. 安装依赖:

```bash
npm install
```

2. 启动桌面应用（会同时启动内置网页服务）:

```bash
npm start
```

3. 单独运行网页服务（如果需要）:

```bash
npm run web
```

数据目录: 项目根目录的 `data/`，包含上传的文件和 `items.json` 元数据（本原型会自动创建）。

注意与后续计划:
- 该原型实现基础 API 和桌面 UI，尚未实现复杂的剪切后跨设备粘贴删除逻辑、缩略图缓存或高速自定义传输协议（scp-like）。可在后续迭代中加入 P2P 传输层、性能优化与更丰富的前端交互。
